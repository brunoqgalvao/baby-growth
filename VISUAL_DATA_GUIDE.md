## Example: Finding Data for 8-month-old

```typescript
// CDC Data (already in file)
import { CDC_BOYS_WEIGHT, getLMSForAge } from './growth-chart-lms-data';

// Exact match in CDC data
const lms = getLMSForAge(CDC_BOYS_WEIGHT, 8.5);
console.log(lms);
// Output: { month: 8.5, L: -0.1334091, M: 9.081119817, S: 0.113217163 }

// Interpolation for 8 months (between 7.5 and 8.5)
const lms8 = getLMSForAge(CDC_BOYS_WEIGHT, 8);
console.log(lms8);
// Output: Interpolated values between 7.5 and 8.5 month data points
```

## Coverage Gaps & Solutions

```
Gap: WHO data 24-60 months
Solution: Download Excel files from WHO
Time: 10 minutes to download and parse

Gap: Data points between months
Solution: Use getLMSForAge() - automatically interpolates

Gap: Different reference standards
Solution: Both CDC and WHO data provided - choose based on:
  - Age range needed
  - Population (US vs International)
  - Feeding method (formula vs breastfed)
```
